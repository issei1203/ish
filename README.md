# ish 
自作なんちゃってシェル

以下のコマンド実装
"ls" ファイルとディレクトリ一覧表示
"cd"　子ディレクトリに移動
"cpd" 親ディレクトリに移動する用のコマンド
"mkdir" ディレクトリの作成
"rmdir" ディレクトリの削除
"touch" ファイルの作成
"rm"　ファイルの削除
"write"　ファイルの中身に書き込む
"read"　ファイルの中身を標準出力する

# 　実行環境
Go 1.20.3

#実行方法
shell.go があるディレクトリで以下のコマンドで実行

```
go run shell.go
```

#ビルド方法
shell.go があるディレクトリで以下のコマンドで実行

```
go build shell.go
```
